# Slot Machine Game

This is a simple slot machine game built in Unity where the player spins reels and wins when all symbols match.  (like yeah we all know, this is how it works)

To run the WebGL build, open Build/WebGL/index.html in a browser or run it using a local server.  (Again I am not teaching you, I know that you know everything)

The project uses coroutines for reel spinning and random number generation for symbol selection. (Again)

The focus was on clean structure, modular scripts, and basic UI interaction. (You know, if you know)
